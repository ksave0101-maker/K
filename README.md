# Korea Management System

한국 관리 시스템 - 통합 비즈니스 관리 플랫폼

## 🚀 주요 기능

- **사용자 관리**: 직원 및 사용자 계정 관리
- **문서 관리**: 문서 및 보고서 관리 시스템
- **통계 및 분석**: 데이터 분석 및 리포팅
- **재고 관리**: 제품 및 재고 추적
- **위치 관리**: 지점 및 위치 정보
- **알림 센터**: 시스템 알림 및 공지사항

## 🛠 기술 스택

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Charts**: Recharts

## 📦 설치 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행 (포트 3002)
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
```

## 🌐 포트 설정

이 시스템은 기본적으로 **포트 3002**에서 실행됩니다.

- 개발 서버: http://localhost:3002
- K Energy Save Dashboard는 포트 3001에서 실행됩니다

## 📁 프로젝트 구조

```
korea-management-system/
├── app/                 # Next.js App Router
│   ├── layout.tsx      # 루트 레이아웃
│   ├── page.tsx        # 홈 페이지
│   └── globals.css     # 글로벌 스타일
├── components/         # React 컴포넌트
├── lib/               # 유틸리티 및 헬퍼 함수
├── public/            # 정적 파일
└── package.json       # 프로젝트 설정
```
